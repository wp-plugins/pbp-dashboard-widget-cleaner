== Description ==
This plugin Delete Wordpress Dashboard Widget

Very Lightweight- Only 2KB

Installation very easy, just install & enjoy.

Created By: www.projoktibangla.net


== Installation ==

= Automatic =

* In the admin panel under plugins page, click Add New
* go to Upload tab
* browse "PBP Dashboard widget cleaner" and click install now
* Click Active plugin
* Enjoy!

= Manual =

* Extract zip file to your wp-content/plugins directory.
* In the admin panel under plugins, activate "PBP Dashboard widget cleaner".
* Enjoy!
